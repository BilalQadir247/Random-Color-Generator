# 🎨 Random Color Generator

A lightweight and interactive web application that generates random HEX color codes with a single click. This project demonstrates dynamic **DOM manipulation** and responsive design principles.

---

### 🌟 Features
* **Instant Color Generation:** Generates a unique HEX code every time the button is clicked.
* **Live Preview:** The background color of the entire page updates instantly to match the generated code.
* **HEX Display:** Shows the exact color code on the screen for easy reference.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
* **Smooth Transitions:** Uses CSS transitions for a professional look when colors change.

---

### 🛠️ Technologies Used
* **HTML5:** For the structural layout.
* **CSS3:** For modern styling, Flexbox centering, and animations.
* **JavaScript (Vanilla):** For the random logic and real-time UI updates.

---

### 🚀 How to Use
1.  Open `index.html` in any web browser.
2.  Click the **"Click Me"** button.
3.  Watch the background change and copy the HEX code displayed on the screen!
